1 Integral of \(x^7\log(x^2) + \log(x^2)e^x\)

Evaluate the integral:

\[ \int \left( x^7 \log(x^2) + \log(x^2) e^x \right) dx \]

Solution: Split into two integrals and solve using integration by parts and special functions.

First integral: \(\int x^7 \log(x^2) dx = \frac{x^8}{4} \log(x) - \frac{x^8}{32} + C_1\)

Second integral: \(\int \log(x^2) e^x dx = 2e^x \log(x) - 2\text{Ei}(x) + C_2\)

\[ \boxed{\frac{x^8}{4} \log(x) - \frac{x^8}{32} + 2 e^x \log(x) - 2 \text{Ei}(x) + C} \]

2 Derivative of \((\cos x)^{\sin x}\)

Find the derivative of \( (\cos x)^{\sin x} \).

Solution: Use logarithmic differentiation:

Let \( y = (\cos x)^{\sin x} \). Then \( \ln y = \sin x \cdot \ln(\cos x) \).

Differentiate: \( \frac{1}{y} \frac{dy}{dx} = \cos x \ln(\cos x) - \sin x \tan x \).

\[ \boxed{\frac{dy}{dx} = (\cos x)^{\sin x} \left( \cos x \cdot \ln(\cos x) - \sin x \cdot \tan x \right)} \]

3 Definite Integral of \(e^x \cos x\)

Evaluate the definite integral:

\[ \int_{-\pi/2}^{\pi/2} e^x \cos x \, dx \]

Solution: Use integration by parts twice to find the antiderivative:

\[ \int e^x \cos x \, dx = \frac{e^x (\sin x + \cos x)}{2} \]

Then evaluate at the bounds.

\[ \boxed{\cosh\left(\frac{\pi}{2}\right)} \]

4 Limit Evaluation

Find the limit:

\[ \lim_{x \to 0} \frac{\tanh\left(\frac{x^2}{2}\right)}{\left(\sin\left(\frac{x}{2}\right)\right)^2} \]

where \(\tanh x = \frac{e^x - e^{-x}}{e^x + e^{-x}}\).

Solution: Use Taylor series expansions:

\(\tanh\left(\frac{x^2}{2}\right) \approx \frac{x^2}{2}\) and \(\left(\sin\left(\frac{x}{2}\right)\right)^2 \approx \frac{x^2}{4}\)

The ratio approaches \(2\).

\[ \boxed{2} \]

5 Distance from Point to Plane

Compute the distance from point \( \mathbf{x} = (2, 0, 1, -3) \) to the plane:

\[ 3x_1 - 2x_2 + 2x_3 - 4x_4 = 1 \]

Solution: Use the formula for distance from a point to a hyperplane:

\[ \text{Distance} = \frac{|\mathbf{a} \cdot \mathbf{x} - b|}{\|\mathbf{a}\|} \]

where \(\mathbf{a} = (3, -2, 2, -4)\) and \(b = 1\).

\[ \boxed{\frac{19\sqrt{33}}{33}} \]

7 Maximal Eigenvalue of \(A^3\)

Let \(A\) be the matrix:

\[ A = \begin{pmatrix} 4 & 1 & 1 \\ 1 & 2 & 3 \\ 1 & 3 & 2 \end{pmatrix} \]

Find the maximal eigenvalue of \(A^3\).

Solution: Find eigenvalues of \(A\): \(\lambda = 3, 6, -1\).

Then eigenvalues of \(A^3\) are \(\lambda^3 = 27, 216, -1\).

\[ \boxed{216} \]

8 Rank Lower-Bound for Modified Matrix

Let \(A \in \mathbb{R}^{n \times n}\) be a full rank matrix. If you are allowed to modify any \(m < n\) entries of \(A\), what is the achievable lower-bound on the rank of the modified matrix?

Solution: Each modification can reduce the rank by at most 1, so the minimum possible rank is \(n - m\).

Example: Zero out \(m\) diagonal entries of the identity matrix.

\[ \boxed{n - m} \]

9 Expected Maximum of Uniform Variables

Let \(X_i\) be i.i.d. uniform random variables on \([0,1]\). Find the mean value of \(Y = \max\{X_1, X_2, \dots, X_{2025}\}\).

Solution: CDF of \(Y\) is \(F_Y(y) = y^{2025}\), so PDF is \(f_Y(y) = 2025y^{2024}\).

Expected value is \(\int_0^1 y \cdot 2025y^{2024} dy = \frac{2025}{2026}\).

\[ \boxed{\frac{2025}{2026}} \]

13 Expected Number of Triangles in Random Graph

Any two nodes in a graph with \(n=10\) nodes are connected with probability \(p=1/4\). Find the expectation of the number of triangles.

Solution: Each triple of nodes forms a triangle with probability \(p^3\). Number of possible triangles is \(\binom{10}{3}\).

14 Vector Counting Problems

Let \( \mathbf{v} = (v_1, v_2, \dots, v_n) \) where \(v_i \in \{0, 1, \dots, q-1\}\). Find:

1. Total number of such vectors

3. Number of vectors differing from the zero vector in at most \(t\) positions

Solution: Basic combinatorial counting.

1. \(\boxed{q^n}\)

2. \(\boxed{\binom{n}{t} (q-1)^t}\)

15 Recurrence Solution

Given the recurrence:

\[ a_{n+1} = \frac{a_n}{1 + n a_n}, \quad n \geq 1 \]

with \(a_1 = 1\), find \(a_n\) in closed form.

Solution: Take reciprocal: \(b_n = 1/a_n\) gives \(b_{n+1} = b_n + n\). Solve to get \(b_n = 1 + \frac{(n-1)n}{2}\).

\[ \boxed{a_n = \frac{2}{n^2 - n + 2}} \]

16 Python GCD Program

Consider this Python program:

func = lambda a, b: a if b == 0 else func(b, a%b)
a = 144
b = 336
print(func(a,b))

What will be the output?

Solution: This implements the Euclidean algorithm for GCD.

The function computes \(\gcd(144, 336) = 48\).

\[ \boxed{48} \]
