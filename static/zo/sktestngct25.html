Calculus
Problem 1. Find derivative with respect to \(x\) of the function \(f(x) = x^{\cos^2 x}\).
Problem 2. Find the indefinite integral \(\int \sqrt{x} e^{-\sqrt{x}} dx\).
Problem 3. Find the following limit \(\displaystyle \lim_{x \to 0} \frac{1-e^{x^2}}{1-\cos x}\).
Problem 4. Evaluate \(\displaystyle \int \left( x^7 \log(x^2) + \log(x^2) e^x \right) dx\).
Problem 5. Find the derivative of \((\cos x)^{\sin x}\).
Problem 6. Compute \(\displaystyle \int_{-\pi/2}^{\pi/2} e^x \cos x \, dx\).
Problem 7. Find \(\displaystyle \lim_{x \to 0} \frac{\tanh\left(\frac{x^2}{2}\right)}{\left(\sin\left(\frac{x}{2}\right)\right)^2}\) where \(\tanh x = \frac{e^x - e^{-x}}{e^x + e^{-x}}\).
Problem 8. Given \(a_{n+1} = \frac{a_n}{1 + n a_n}\) with \(a_1 = 1\), find \(a_n\) in closed form.
Linear algebra
Problem 9. Compute the distance from point \(\mathbf{x} = (1, -2, 0, 3)\) to the plane \(x_1 + 3x_2 - x_3 + x_4 = 3\).
Problem 10. Let \(\mathbf{A}, \mathbf{B}\) and \(\mathbf{C}\) be \(n \times n\) invertible matrices. Simplify the following expression \[ \mathbf{C}^{-1} (\mathbf{A}^2 \mathbf{B}^{-1})^{-3} (\mathbf{C} \mathbf{A}^{-3})^{-2} \mathbf{C}^2. \]
Problem 11. Let \(A, B, C\) be \(n \times n\) invertible matrices. Simplify \(C^{-1}(A^2B^{-1})^{-2}A^{-2}(CA^{-3})^{-2}(CB^{-1})^2\).
Problem 12. Let \(A = \begin{pmatrix} 4 & 1 & 1 \\ 1 & 2 & 3 \\ 1 & 3 & 2 \end{pmatrix}\). Find the maximal eigenvalue of \(A^3\).
Problem 13. Let \(A \in \mathbb{R}^{n \times n}\) be a full rank matrix. If you are allowed to modify any \(m < n\) entries of \(A\), what is the achievable lower-bound on the rank of the modified matrix?
Problem 14. Find the dimension of the space of polynomials of 4 variables with degree not greater than 3.
Probability and Combinatorics
Problem 15. Let \(X_i\) be i.i.d. uniform random variables on \([0,1]\). Find the mean value of \(Y = \max\{X_1, X_2, \dots, X_{2025}\}\).
Problem 16. A fair coin is tossed repeatedly. Find \(E[l(HH)]\) and \(E[l(HT)]\), where \(l(X)\) is the number of tosses until the first appearance of pattern \(X\).
Problem 17. A traveler starts in city A. Each day: in A, stays with probability 1/3, goes to B or C with probability 1/3 each; in B, goes to A or C with probability 1/2 each. Find the expected number of days to get from A to C.
Problem 18. In a graph with \(n = 10\) nodes, each pair is connected with probability \(p = 1/4\). Find the expected number of triangles.
Problem 19. Let \(\mathbf{v} = (v_1, \dots, v_n)\) where \(v_i \in \{0,1,\dots,q-1\}\).
(a) Total number of such vectors
(b) Number differing from \(\mathbf{0}\) in exactly \(t\) positions
(c) Number differing from \(\mathbf{0}\) in at most \(t\) positions
